# 📇 Connectly - Contact Management Application

A modern and responsive contact management application built with React, Vite, and Tailwind CSS.

Connectly allows users to add, edit, delete, and search contacts through a beautiful glassmorphism-inspired user interface.

---

## 🚀 Live Demo

🔗 https://sk-pandey.github.io/Contacts/

---

## 📸 Preview

Connectly provides a clean and elegant interface for managing personal contacts with:

- Add new contacts
- Edit existing contacts
- Delete contacts
- Real-time search functionality
- Responsive design
- Modern glassmorphism UI
- Form validation
- Dynamic avatars
- Smooth animations

---

## ✨ Features

### Contact Management

- Add contacts with:
  - Name
  - Email
  - Phone Number

### Validation

- Required field validation
- Email format validation
- Phone number validation (10 digits)

### Search

- Search contacts by:
  - Name
  - Email
  - Phone Number

### Edit & Delete

- Update contact information
- Remove contacts instantly

### UI/UX

- Modern dark theme
- Glassmorphism cards
- Animated background orbs
- Responsive layout
- Interactive hover effects
- Dynamic avatar generation

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Sk-Pandey/Contacts.git
```

Move into the project directory:

```bash
cd Contacts
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

---

## 🏗️ Build Project

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

---

## 🚀 Deploy to GitHub Pages

Deploy the application:

```bash
npm run deploy
```

The application will be published automatically to GitHub Pages.
